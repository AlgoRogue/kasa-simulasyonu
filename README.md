# Kasa Simulasyonu

Market kasa yazılımı - Programlama dersi projesi

## Özellikler

- Ürün stok takibi
- Satış işlemleri
- Fiş yazdırma
- Otomatik stok kontrolü

## Gereksinimler

- C Compiler (GCC önerilen)
- CMake 3.10 veya üstü

## Kurulum

```bash
mkdir build
cd build
cmake ..
make
```

## Kullanım

```bash
./kasa_simulasyonu
```

## Test

```bash
./run_tests
```
